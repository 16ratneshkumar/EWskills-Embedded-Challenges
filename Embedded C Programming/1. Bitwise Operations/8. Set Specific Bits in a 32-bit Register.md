### ğŸ’¡ Hint

To set multiple bits to 1 without changing other bits:

1. Create a **bitmask** with `len` bits set to 1:

   ```c
   (1U << len) - 1
   ```
2. Shift that mask to the target position:

   ```c
   ((1U << len) - 1) << pos
   ```
3. Use **bitwise OR** to set the bits:

   ```c
   reg | (((1U << len) - 1) << pos)
   ```
