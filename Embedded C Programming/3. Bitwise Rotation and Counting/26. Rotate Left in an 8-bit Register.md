* Rotating left `n` positions in an 8-bit value:

  ```c
  (reg << n) | (reg >> (8 - n))
  ```

* Use `n % 8` to avoid undefined behavior if `n >= 8`

* Cast the result to `uint8_t` to ensure it remains within 8 bits:

  ```c
  return (uint8_t)((reg << n) | (reg >> (8 - n)));
  ```
